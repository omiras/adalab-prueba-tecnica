# PokeAPI Cliente Prueba Técnica AdaLab

## Setup

`npm install`
`npm run dev`

Si se quiere usar la API desarrollada para la prueba técnica, copiar el fichero **.env.example** en el fichero **.env** para tener en cuenta la variable de entorno
